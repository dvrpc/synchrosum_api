SynchroSum

File names must meet the following criteria:

  • Synchro reports must be text files (.txt)
  • Simtraffic reports must be pdfs (.pdf)
  • Tool will only accept one of each file type (.pdf, .txt)
  • You can upload just one file if you'd like (either Synchro or Simtraffic), or both the PDF and the text file.

      • Summary file download takes a few seconds after pressing submit..
